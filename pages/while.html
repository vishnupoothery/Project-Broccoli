While loop
A while loop repeats a block of code as long as a certain condition is true.
Syntax of while loop
1. test_variable = value
2. while(test_condition):
3.      code to be repeated
4.      test_variable += 1
5. #statement not part of while

Eg 1:
1. current_value = 1
2. while current_value <= 5:
3.      print(current_value)

Eg 2:
1. msg = ''
2. while msg != 'quit':
3.      msg = input("What's your message? ")
4.      print(msg)

Example 2 lets the user choose when to exit the program. The above program quits when the user types 'quit'
