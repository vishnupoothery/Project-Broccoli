HelloWorld

Programming tutorials since the beginning of time have started with a little program called "Hello, World!" So here it is:

1. print("Hello World!")

If you are using the command line to run programs then type it in with a text editor, save it as hello.py and run it with python3 hello.py

Output :
Hello World

What we were doing above was using a function called print. The function's name - print - is followed by parentheses containing zero or more arguments. So in this example there is one argument, which is "Hello, World!". Note that this argument is a group of characters enclosed in double quotes (""). This is commonly referred to as a string of characters, or string, for short.
